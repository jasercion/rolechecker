# rolechecker
Simple bot to return list of users with a specified discord role
## setup
Require `config.json` file to be created in the top level directory with the form:
```
{
    "token": "token-value",
    "clientId": "bot-application-id",
    "guildId": "guild-id"
}
```
Requires `discord.js`
To run:
```
node deploy-command.js
node index.js
```
